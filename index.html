Enter your GitHub credentials to access your Obsidian vault.

Security Notice

Your token is stored locally in your browser and never sent to any server except GitHub. Only use this app from trusted sources. Avoid using on shared computers.

What is a Personal Access Token (PAT)?

A PAT is like a password that gives this app permission to read and edit files in your GitHub repository. You create it once and paste it below.

Create a GitHub Token ‚Üí

When creating your token, set these permissions:

Option 1 (Recommended): Fine-grained token

  • Repository access: Only select your vault repository
  • Repository permissions ‚Üí Contents ‚Üí Read and write (both required!)

Option 2: Classic token

  • Check the repo scope checkbox (gives read and write access)
